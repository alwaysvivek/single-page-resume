## 🏗️ Technical Architecture

This portfolio is a high-fidelity, performance-optimized single-page application built with the latest **React 19** and **Vite** ecosystem. It features a custom design system focused on a "premium dark" aesthetic and fluid, physics-based interactions.

### Key Features
- **Premium UI/UX**: Injected grain texture overlay for tactile depth, combined with glassmorphism and subtle micro-animations.
- **Parallax Navigation**: A custom `useScroll` powered header that dynamically morphs (shrinks, blurs, and borders) to stay prominent over content.
- **Command Palette (⌘K)**: A central search interface for rapid internal navigation, project discovery, and quick actions.
- **Signal Skills Matrix**: A curated technical domain map featuring AI Engineering (LangGraph, RAG), Low-Level Systems (RESP, Compilers), and Modern Frontend (React 19).
- **Physics-In-View**: Robust `whileInView` entrance animations for every major section, ensuring a cohesive and dynamic scrolling experience.

## 🚀 Tech Stack
- **Core**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State & Logic**: Custom Hooks (Scroll, Command Search)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Local Development

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/alwaysvivek/single-page-resume.git
   cd single-page-resume
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Environment Setup:**
   Create a `.env` file and add your `GEMINI_API_KEY` (required for agentic features):
   ```bash
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run Development Server:**
   ```bash
   npm run dev
   ```
